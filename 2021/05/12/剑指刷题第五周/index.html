剑指刷题第五周

Posted on 2021-05-12 In Algorithm

2021.5.3~2021.5.9

剑指44 - 数字序列中某一位的数字

思路：先确定是几位数(最多11位)，从一位数开始减，n - 10 - 90 * 2 - 900 * 3 …；第二步确定这个数所属数字的大小；第三步确定属于那个数的第几位

剑指45 - 把数组排成最小的数字

思路：自定义排序方式， a < b 等价于 字符串ab < 字符串ba

离散数学中规定只要满足反对称性和传递性，那就可以排序

  • 反对称性：a <= b, b <= a, 则 a = b
  • 传递性：a < b, b < c, 则a < c

剑指46 - 把数字翻译成字符串

思路：计数问题都可以看做是dp问题

  • 状态表示：f[i]表示前i位数字一共有多少种翻译方式
  • 状态计算：①第i位单独翻译：f[i] = f[i-1]，②第i位和第i-1位一起翻译：f[i] = f[i-2]（两位数必须在0~25之间，且必须合法(不能出现05 06这种情况零开头的情况)）
  • 边界定义：f[0] = 1

剑指47 - 礼物的最大价值 - 经典dp问题

思路：dp问题

  • 状态表示：f[i] [j]表示从左上角开始走到(i, j)位置的最大价值
  • 状态计算：f[i] [j]只能从f[i - 1] [j]和f[i] [j - 1]转移过来，f[i] [j] = max(f[i - 1] [j], f[i] [j - 1]) + gif[i] [j];
  • 边界定义：f[i] [0] = f[0] [j] = 0;

剑指48 - 不含重复字符的最长字符串 - 经典双指针问题

思路：
暴力做法：j从前往后枚举终点，i从0开始往前走，每走过一个字符就将其加入hash map，直到出现重复数字，然后j往前挪一步，最终结果取max，时间复杂度是O(n^2)
优化：可以发现i和j是单调的，即j往后移的时候，j对应的i一定是往后移的，时间复杂度为O(n)

注：满足第二个指针往后移时，第一个指针一定也往后移的条件时，才可以用双指针算法

剑指49 - 丑数

思路：用ai/2,bi/3,ci/5去还原 ai,bi,ci然后三路归并

三指针做法，以2为质因子的ai、以3为质因子的bi和以5为质因子的ci，ai/2,bi/3,ci/5得到的三路序列也恰好是丑数序列，（没搞懂）

剑指50Ⅰ - 字符串中第一个只出现一次的字符

思路：用哈希表存每个字母出现的次数

剑指50Ⅱ - 字符流中第一个只出现一次的字符

思路：如果用上一题的思路，总的时间复杂度变为O(n^2)，想将O(n^2)的时间复杂度优化为O(n)的复杂度，可利用双指针算法或者单调队列算法

单调性：随着字符的加入，答案肯定是从前往后走的，所以可以使用双指针算法

剑指51 - 数组中的逆序对 - 模板题

思路：暴力做法是O(n^2)，可利用归并排序的思想

剑指52 - 两个链表的第一个公共节点

思路：设置两个指针都走两个链表的一圈，第一次相遇处即为公共节点

剑指53Ⅰ - 数字在排序数组中出现的次数

思路：二分

整数二分终极模板

模板1：mid属于左半边 [l, mid] [mid + 1, r]

更新：

  • 答案在左半边：r = mid
  • 答案在右半边：l = mid + 1

模板2：mid属于右半边 [l, mid - 1] [mid, r]

更新：

  • 答案在左半边：r = mid - 1
  • 答案在右半边：l = mid
